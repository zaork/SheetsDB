SheetDB

SheetDB is a powerful tool to export a Google Sheet to an HTML table or JSON array.

Features

  • Custom names for table columns and JSON objects
  • Powerful filtering of unwanted data and empty rows
  • Add URL's to any column/object
  • Directly import JSON into TaffyDB and other JavaScript databases
  • Save configuration across users in a spreadsheet

Sheet to Data will export the active sheet into an HTML table or JSON array. The only constraint is that the top row of the sheet is a header. Right now, Sheet to Data does not see or use those header names. This will be added in the future, along with an option to tell Sheet to Data whether a header row is present.
