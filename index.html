SheetsDB

A powerful tool to use a Google Spreadsheet as a low-effort database.

Download this project as a .zip file Download this project as a tar.gz file

Features

  • Spreadsheet to JSON array
  • Pair a column of URLs with a column of names
  • Custom names for table columns and JSON objects
  • Automatically determine headers using column titles
  • Powerful filtering of unwanted data and empty rows
  • Directly import JSON into many types of databases
  • Save & load configuration for each spreadsheet
  • Generate multiple files at once
  • [Planned] Send files directly to server

SheetsDB will export sheets into a JSON object.
