SheetDB

A powerful tool to convert a Google Spreadsheet to an HTML table or JSON array.

Download this project as a .zip file Download this project as a tar.gz file

Features

  • Custom names for table columns and JSON objects
  • Powerful filtering of unwanted data and empty rows
  • Add URL's to any column/object
  • Directly import JSON into TaffyDB and other JavaScript databases
  • Save configuration across users in a spreadsheet

Sheet to Data will export the active sheet into an HTML table or JSON array.
